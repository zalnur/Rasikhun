# راسخون - اختبار متشابهات القرآن الكريم

تطبيق ويب تفاعلي ومبتكر مصمم خصيصاً لمساعدة حفظة كتاب الله عز وجل على اختبار وتثبيت **متشابهات القرآن الكريم** من خلال أسلوب تفاعلي يعتمد على اختيار التكملة الصحيحة لموضع التشابه.

---

## 🌟 مميزات التطبيق

* 🌓 **دعم كامل للوضع الداكن والوضع المضيء** بشكل جمالي وأنيق مريح للعين أثناء القراءة الطويلة.
* ⚙️ **خيارات اختبار متقدمة ومخصصة**:
  * **تحديد طول الاختبار**: إمكانية اختيار عدد الأسئلة (5، 10، 20، 30 سؤالاً).
  * **تصفية الاختبار حسب السورة**: إمكانية التركيز على سورة معينة أو إدراج جميع السور.
  * **تخصيص الرسم القرآني**: دعم الرسم العثماني والرسم الإملائي القياسي.
  * **آيات السياق**: إمكانية إظهار حتى 3 آيات قبل و/أو بعد موضع التشابه لتسهيل التذكر وفهم السياق.
  * **طريقة عرض الفراغ**: إخفاء الآية المستهدفة بالكامل أو إخفاء مواضع الاختلاف فقط (Diff Mode).
* 📊 **تغذية راجعة فورية ومفصلة**: يعرض التطبيق الآية المقابلة المشابهة وتوجيه موضع الاختلاف فور الإجابة على السؤال.
* 📝 **تقرير النتيجة النهائي**: إحصائيات دقيقة لنسبة النجاح مع سجل كامل بالأسئلة وإجاباتك عليها لتسهيل المراجعة.

---

## 🛠️ التقنيات المستخدمة

تم بناء التطبيق باستخدام تقنيات الويب الأساسية لضمان السرعة والعمل المباشر دون الحاجة لخطوات تثبيت معقدة:
* **الهيكل والواجهة**: HTML5 و Javascript (Vanilla JS).
* **إطار العمل التفاعلي**: Vue.js 3 (عبر CDN).
* **التنسيق والتصميم**: Tailwind CSS (عبر CDN) مع نظام ألوان مخصص وتأثيرات زجاجية (Glassmorphism) وتأثيرات حركية سلسة.
* **الأيقونات**: Lucide Icons.
* **المؤثرات التفاعلية**: Canvas Confetti للمؤثرات الحركية عند الإجابة الصحيحة.

---

## 🚀 طريقة التشغيل

بما أن التطبيق يعتمد على ملفات ويب ثابتة (Static Web Pages)، يمكنك تشغيله بعدة طرق سهلة:

### 1. التشغيل المباشر عبر المتصفح
فقط قم بفتح ملف `index.html` في أي متصفح ويب حديث.

### 2. التشغيل عبر خادم محلي (Local Server)
إذا كنت تستخدم بيئة مثل XAMPP أو WampServer:
1. ضع مجلد المشروع داخل المجلد المخصص للمواقع (مثال: `C:\xampp\htdocs\Rasikhun`).
2. افتح المتصفح وانتقل إلى العنوان: `http://localhost/Rasikhun`.

## 📂 هيكل المجلدات الرئيسي

* `index.html` - الصفحة الرئيسية للواجهات وعرض الاختبار.
* `css/style.css` - ملف الأنماط والتأثيرات البصرية الخاصة بالتصميم.
* `js/app.js` - منطق التطبيق التفاعلي وإدارة الحالة بـ Vue 3.
* `js/diffEngine.js` - محرك المقارنة والتحقق من الفروقات وتوليد خيارات الأسئلة تلقائياً.
* `data/` - يحتوي على ملفات البيانات النصية للقرآن الكريم والمتشابهات:
  * `similarities.json` - ملف يحتوي على مجموعات متشابهات الآيات وتوجيهاتها.
  * `quran_text.js` - نص القرآن الكريم المولد لعرض الآيات والسياق.
* `build_quran.js` - سكربت Node.js لتحديث وبناء بيانات القرآن الكريم من ملفات المصدر.
